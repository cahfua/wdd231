Samoa Dog Care & Adoption – Site Plan

Individual Website Project – WDD 231

1. Site Name

Site Name: Samoa Dog Care & Adoption

Reason for Name: The name explains the purpose clearly: helping people in Samoa learn dog care basics and explore adoptable dogs that need safe homes.

2. Site Purpose

The site will provide:

  • Simple dog care guides tailored to Samoa’s needs.
  • A dynamic page showing adoptable dogs from a JSON data source.
  • Helpful steps for new and current dog owners.

This project helps me practice web development while creating something meaningful for my community.

3. Scenarios

  • Scenario 1: “Where can I see adoptable dogs in Samoa?”
  • Scenario 2: “How do I care for my dog properly?”
  • Scenario 3: “Can I quickly compare dog ages, sizes, and personalities?”

4. Color Schema

These colors match the soft lavender graphic identity in my project proposal.

Primary: #7F6BAE (Medium Lavender)

Used for header, footer, key headings, and important UI elements.

Accent: #C7B7E4 (Light Lavender)

Used for subtle highlights, borders, and decorative accents.

Neutral: #EDE6D6 (Soft Cream)

Used as the main page background behind white content cards.

Note: These colors are used in this document to satisfy the color schema requirement.

5. Typography

These fonts match the graphic identity image.

Body Font

Lato – used for all paragraphs, navigation text, and general content.

Heading Font

Poppins – used for the site title and all section headings.

Both fonts are used in this site-plan document through the CSS rules above.
